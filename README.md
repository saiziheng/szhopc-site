# szhopc-site

`business.szhopc.cn` personal IP hub for szh: portfolio, build in public updates, OPC practice identity, and trusted contact entry.

## Stack

- Next.js App Router
- TypeScript
- Tailwind CSS
- Static export (`next.config.ts` uses `output: "export"`)
- System font stack only, no `next/font/google`

## Commands

```bash
npm run typecheck
npm run lint
npm run build
npm run test:e2e
```
